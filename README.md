# Lists — Linked List Data Structure Implementation 📋

A from-scratch implementation of linked list data structures in C++,
built without relying on the STL. Covers both singly and doubly linked
lists with full insert, delete, search, and traversal operations.

## Overview
This project implements core list data structures from the ground up,
reinforcing a deep understanding of how memory, pointers, and dynamic
allocation work under the hood — the fundamentals that power every
higher-level data structure in modern software.

## Features
- Custom node and list class implementations
- Insert at head, tail, and arbitrary index
- Delete by value and by position
- Search and traversal operations
- Memory management with proper allocation and deallocation
- Makefile for streamlined compilation

## Concepts Demonstrated
- Pointer manipulation and dynamic memory allocation in C++
- Linked list architecture — nodes, head/tail pointers, chaining
- Class design with constructors and destructors
- Manual memory management — avoiding leaks with proper deallocation
- Build automation with Makefile

## How to Run
```bash
make
./lists
```

## Language
- **C++** with Makefile build system
